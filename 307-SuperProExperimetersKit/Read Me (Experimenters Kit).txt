2) INSTALLING THE HITECHNIC SuperPro Sensor BLOCK FOR LEGO MINDSTORMS NXT SOFTWARE

Administrator privileges are required when installing the SuperPro Sensor Block for LEGO MINDSTORMS NXT Software.

1. Verify that all installed versions of LEGO MINDSTORMS NXT Software are not currently running.

2. Launch LEGO MINDSTORMS NXT Software and create a new program by clicking 'Go' under 'Start New Program'. It is preferred that you haven't opened any other programs (during this session of the software) prior to importing blocks.

3. Under the 'Tools' menu, select the 'Block Import and Export Wizard...'.

4. Click 'Browse' to select the directory ("SuperPro Sensor") that contains the SuperPro Sensor Block.

5. Once the wizard has found the blocks, select the block by clicking on it.

6. Click 'Import'

7. Close.


3) UNINSTALLING THE HITECHNIC Prototype Sensor BLOCK FOR LEGO MINDSTORMS NXT SOFTWARE

The Block Import/Export Utility doesn't have the ability to uninstall a block.

To completely uninstall a block from the LEGO MINDSTORMS NXT Software installation, you will need to manually delete the files that were created during the import process.

Before you begin, verify that all installed versions of LEGO MINDSTORMS NXT Software are not currently running.

The SuperPro Sensor Block is contained in the directory “SuperPro Sensor”.

The Block can be removed from the palette by deleting the palette file for the block found under

	<LEGO MINDSTORMS NXT Directory>/engine/EditorVIs/BlockRegistry/<Palette Name>/SuperPro Sensor.txt

The actual block files can be deleted at

	<LEGO MINDSTORMS NXT Directory>/engine/vi.lib/LEGO/Blocks/SuperPro Sensor

The help files can be deleted at

	<LEGO MINDSTORMS NXT Directory>/engine/EditorVIs/Help Content/Block_SuperPro Sensor.htm

	<LEGO MINDSTORMS NXT Directory>/engine/EditorVIs/Help Content/Full Help/topics/Help_SuperPro Sensor.htm

	<LEGO MINDSTORMS NXT Directory>/engine/EditorVIs/Help Content/SuperPro Sensor Files

	<LEGO MINDSTORMS NXT Directory>/engine/EditorVIs/Help Content/Full Help/topics/SuperPro Sensor Files
